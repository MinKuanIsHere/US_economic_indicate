# US_economic_indicate

美國的經濟指標一直是世界的焦點，包含聯準會(Fed)的利率、CPI、核心CPI、非農就業人數、道瓊指數。
可藉由這五個指標繪製時間序列的線圖，並用相關係數觀察。
